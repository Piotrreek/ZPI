# Projekt SRS – Specyfikacja Wymagań Oprogramowania

To repozytorium zawiera dokumentację projektu realizowanego w ramach przedmiotu ZPI.

Przygotowanie kompletnej Specyfikacji Wymagań Oprogramowania (SRS) dla wybranego systemu informatycznego (System Zarządzania Cyklem Życia Pracownika: Strategie Rekrutacji i Rozwoju).

Opracowana specyfikacja znajduje się w pliku `SRS.md`.

Dokumentacja zawiera m.in. opis celu, wizji, funkcjonalności, wymagań oraz analizę porównawczą zgodnie z wytycznymi zadania.
